# Документация для стейкхолдеров

Отдельный репозиторий с материалами для заказчиков и стейкхолдеров проекта **BSFY** — системы ежедневных библейских активностей для христианской молодежи.

## Назначение

Здесь хранятся документы, которые передаются заказчику: описание функционала, ценность продукта, пользовательские сценарии. Техническая и внутренняя документация (схема БД, спецификации для разработки) находится в основном репозитории проекта.

## Содержимое

| Документ | Описание |
|----------|----------|
| [mvp_scope_ru.md](mvp_scope_ru.md) | **MVP (первый этап):** что входит в минимальный набор возможностей, что отложено; без технических терминов |
| [mvp_estimate.md](../prds/mvp_estimate.md) | Оценка трудозатрат и стоимости MVP (в каталоге prds; адаптирована для нетехнического читателя) |

| Документ | Примечание |
|----------|------------|

## Для кого этот репозиторий

- Заказчик проекта
- Руководство церкви / объединения
- Лица, принимающие решение о внедрении

## Связь с основным проектом

Этот репозиторий может использоваться как самостоятельный (только документы) или подключаться к основному репозиторию BSFY. Обновление документов здесь не требует доступа к коду приложения.
